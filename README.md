## Setup using Docker

1. Clone the repository
2. Run `docker-compose up`
3. Visit `http://localhost:3001`

## Features Supported:
- Login
- Logout
- View Products
- View Product Details
- Buy Product
- Add to Cart
- Wishlist

## Tech Stack 
- Next.js, TS, React
- Tailwind CSS
- MongoDB
- Express
- Node.js
- Vercel
