# Spotify Clone

## Setup using Docker

1. Clone the repository
2. Run `docker-compose up`
3. Visit `http://localhost:3000`

## Features Supported:
- Auth
- Search Song
- Play Song
- Favourite

## Tech Stack 
- Next.js, TS, React
- Tailwind CSS
- MongoDB
- Express
- Node.js
- Vercel
 
Demo: [Spotify Clone](https://spotify-clone-psidh.vercel.app/)
